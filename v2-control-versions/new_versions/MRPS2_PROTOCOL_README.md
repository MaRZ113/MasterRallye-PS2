# MRPS2 protocol bundle

This folder contains the reproducibility guardrails for the Master Rallye PS2 unpacker project.

Files:
- `MRPS2_CANONICAL_INPUT.json` — canonical input hashes and source parts
- `MRPS2_OFFICIAL_CHECKPOINT_v210.json` — current official baseline
- `MRPS2_DOMAIN_REGISTRY_v210.json` / `.csv` — snapshot of domain rulepacks on disk versus official checkpoint membership
- `MRPS2_MERGE_PROTOCOL_v1.md` — workflow that defines when a domain becomes official
